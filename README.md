# Mithron Flight Systems

Next.js storefront and admin platform for Mithron drone systems.

## Quick start

```bash
npm install
npm run dev
```

Open http://127.0.0.1:3000

## Scripts

```bash
npm run dev
npm run build
npm run start
npm test
```

Copy `.env.example` to `.env.local` and fill in Supabase credentials before running.
